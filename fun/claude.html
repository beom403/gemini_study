Chatbot 요약

  • 질문-응답 패턴에 최적화
  • 사람이 모든 단계를 안내해야 함
  • 빠르고 예측 가능한 결과
  • 외부 세계와 상호작용 불가
  • 도구 없이 언어만으로 작동

AI Agent 요약

  • 목표만 주면 경로를 스스로 결정
  • 계획 → 실행 → 관찰 루프 반복
  • 파일·코드·웹 등 도구 자율 활용
  • 실패 시 전략을 수정하며 재시도
  • 복잡한 장기 작업 처리 가능
